# 🛡️ Telegram Account Diagnostic Tool

CLI-інструмент на базі Telethon для швидкої перевірки безпеки акаунту Telegram, моніторингу активних сесій та управління повідомленнями.
Натхненний подіями реєстрації zero-click вразливостей і небажанням користуватись GUI-клієнтами. Інструмент не претендує конкурувати з вже росповсюдженими tg-cli.

## ✨ Можливості
- **Аналіз сесій:** Відправка команди на завершення сесій, що не входять у ваш "білий список".
- **Статус акаунту:** Швидкий перегляд ID, статусу онлайн та кількості підключених пристроїв.
- **Менеджер повідомлень:** Читання та відповідь на непрочитані повідомлення через консоль.

## 🚀 Встановлення та запуск

### 1. Клонування репозиторію
```bash
git clone https://github.com/iliapavlov/teleguard-cli.git
cd teleguard-cli
```

### 2. Налаштування середовища

Створіть віртуальне середовище та встановіть залежності:
```bash
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Конфігурація

Створіть файл .env у кореневій папці проекту:

```bash
TG_API_ID=ваш_id
TG_API_HASH=ваш_hash
TG_PHONE=+380XXXXXXXXX
```

Отримати API ID та Hash можна на my.telegram.org.

#### ⚙️ Налаштування Whitelist (configuration.json)

**Крок 1:** Запустіть перевірку статусу:
```bash
python main.py --status
```

**Крок 2:** Скопіюйте запропонований рядок із виводу консолі та додайте його у файл:
```json
{
    "allowed_sessions": [
        "PC 64bit|Telegram Desktop",
        "iPhone 15|Telegram iOS"
    ]
}
```
**Крок 3:** Запустіть очистку:
```bash
python main.py --clean
```

### 4. Використання

Запустіть скрипт із потрібними прапорцями:

    - Перевірити статус: python main.py --status
    - Очистити сесії: python main.py --clean
    - Непрочитані: python main.py --unread

### ⚠️ Важливо
.
Не використовуйте для облікових записів без двофакторної авторизації.
